Most AI tools will agree with whatever you say. That's not help — that's expensive validation.

I built **cursor-llm-council**: an open-source Cursor skill that runs a multi-model AI council before you ship bad ideas.

How it works:
→ 4 independent reviewers on different models (Claude, Gemini, GPT)
→ Mandatory devil's advocate — always included
→ Pre-flight cost disclosure (no surprise token bills)
→ Supervisor synthesis with a "do not do" list
→ Honesty charter that bans hollow praise like "great idea"

It's prompt-enforced honesty, not magic. The skill tells you that upfront.

Install in 30 seconds:
```
git clone https://github.com/Siddarthb07/cursor-llm-council.git
cd cursor-llm-council
./install.sh   # or .\install.ps1 on Windows
```

Then in Cursor: `/council Should I rewrite everything in Rust?`
